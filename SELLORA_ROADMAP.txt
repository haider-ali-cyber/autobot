2. AUTHENTICATION SYSTEM
   - NextAuth.js integration
   - Email/password login (bcrypt for passwords)
   - Google OAuth login
   - JWT session handling
   - Protected routes middleware
   - Password reset flow (email based)

3. API ROUTES (Next.js App Router)
   - /api/auth/* (NextAuth handles most)
   - /api/user/profile (GET, POST, PUT)
   - /api/user/settings (notifications, preferences)
   - /api/stores (CRUD + connect/disconnect)
   - /api/products (CRUD + import from stores)
   - /api/orders (fetch from connected stores)

4. STORE CONNECTIONS (Read-Only APIs)
   - Amazon SP-API connection:
     * OAuth flow for seller authorization
     * Fetch orders (last 30/60/90 days)
     * Fetch inventory levels
     * Fetch product catalog
   
   - Shopify Admin API:
     * OAuth/app installation flow
     * Fetch orders
     * Fetch products
     * Webhook setup for real-time updates

5. DATA SYNC JOBS
   - Background sync every 15 minutes
   - BullMQ + Redis for job queue
   - Sync: Orders, inventory, revenue
   - Error handling and retry logic

TECH STACK FOR PHASE 2:
- Backend: Next.js API Routes (no separate server needed)
